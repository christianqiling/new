# 店内计时收费系统

一个**零依赖**的网页版店内计时收费系统：顾客自助进店、按进店时长（分时段单价）实时扣费，管理员人工充值、分时段定价、查看营业报表。全部基于 Node.js 内置模块（`node:http` + `node:sqlite` + `node:crypto`），**无需 `npm install`**，一条命令即可运行。

## 功能

**顾客**
- 自助注册（用户名 + QQ 号 + 密码）、登录
- 进店 / 离店，按当前时段单价**实时扣费**，余额耗尽自动下线
- 实时查看账户余额、本次时长与费用
- 查看店内当前在店**顾客人数**、在店**管理员姓名**与价格表（看不到其他顾客姓名，保护隐私）

**管理员（分管理员 / 总管理员）**
- 进店 / 离店打卡（不计费）
- 查看当前在店**全部人员真实姓名**
- 按权限开放的功能：
  - **为用户充值**（人工录入）
  - **调整营业价格**（分时段设置"几点–几点 = X 元/小时"）
  - **查看营业报表**：当日营业额、当日进出/时长记录、近 7 天营业额柱状图
- **总管理员**：将用户设为分管理员，并逐项分配权限

## 运行

```bash
./start.sh
# 或指定端口
PORT=8080 ./start.sh
```

然后在浏览器打开 `http://localhost:3000`。

> 注意：本沙箱环境存在一个无效的 `NODE_OPTIONS` 预加载，`start.sh` 已自动 `unset NODE_OPTIONS`。若直接用 node 运行，请使用：`env -u NODE_OPTIONS node src/server.js`。

## 默认账号

首次启动会自动创建总管理员：

- 用户名：`admin`
- 密码：`admin123`

**请登录后尽快修改**（可通过环境变量 `SUPER_ADMIN_USER` / `SUPER_ADMIN_PASS` 覆盖初始值）。默认时价为全天 10 元/小时，可在"营业定价"中修改。

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `3000` | 服务端口 |
| `DB_PATH` | `data/store.db` | SQLite 数据库文件路径 |
| `TICK_MS` | `60000` | 实时扣费结算间隔（毫秒） |
| `SUPER_ADMIN_USER` | `admin` | 初始总管理员用户名 |
| `SUPER_ADMIN_PASS` | `admin123` | 初始总管理员密码 |

## 目录结构

```
src/
  server.js    HTTP 服务器、路由、静态资源、扣费定时器
  api.js       各接口处理逻辑
  db.js        SQLite 初始化、建表、种子数据
  auth.js      密码哈希、会话、角色与权限
  billing.js   分时段计费引擎（实时扣费、余额耗尽结算）
public/
  index.html / app.js / styles.css   纯原生前端单页应用
data/
  store.db     运行时自动生成的数据库
```

## 计费说明

- 货币以"分"为单位存储，避免浮点误差，展示时换算为元。
- 进店要求余额 > 0；游玩中每 `TICK_MS` 结算一次并写入流水；前端每秒插值让数字实时跳动。
- 跨时段游玩按各时段费率分段计费；未覆盖的时段视为免费。
- 余额耗尽时在耗尽时刻自动结束并标记"余额耗尽"。
